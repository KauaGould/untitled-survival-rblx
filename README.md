Untitled Survival Game é um projeto de jogo de sobrevivência desenvolvido no Roblox utilizando Luau.

O projeto conta com sistemas de inventário, vida, fome, sede, coleta de recursos e ciclo de dia e noite em um ambiente low-poly.

O objetivo do jogo é sobreviver coletando recursos como madeira e pedra enquanto explora o mapa.

Projeto desenvolvido para fins de aprendizado e prática em desenvolvimento de jogos.
